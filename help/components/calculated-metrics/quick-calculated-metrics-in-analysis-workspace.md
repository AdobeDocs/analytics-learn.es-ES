# [!UICONTROL Métricas calculadas] de forma rápida en Analysis Workspace {#quick-calculated-metrics-in-analysis-workspace}

Ahora puede crear [!UICONTROL métricas calculadas] sobre la marcha dentro de Analysis Workspace.  Si desea realizar un cambio o una división porcentual de forma rápida con algunas [!UICONTROL métricas] en la [!UICONTROL tabla de forma libre], solo tiene que seleccionar las [!UICONTROL métricas] en el encabezado, hacer clic con el botón secundario y elegir el operador que desee.  La nueva [!UICONTROL métrica calculada] aparecerá a la derecha.
