# Visualización de líneas en Analysis Workspace

La visualización Línea ilustra las métricas a lo largo de un período de tiempo de tendencias. La configuración de visualización permite personalizar el aspecto visual de cada línea e incluye ajustes para Mostrar ejes X/Y, Mostrar etiquetas mín/máx y Mostrar línea de tendencia.
