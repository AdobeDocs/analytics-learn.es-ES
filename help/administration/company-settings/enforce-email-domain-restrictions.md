# Aplicar restricciones de dominio de correo electrónico

El Administrador de seguridad permite controlar el acceso a los datos de informes. Las opciones incluyen contraseñas seguras, caducidad de las contraseñas, restricciones de inicio de sesión de direcciones IP y restricciones de dominio de correo electrónico. Aplicar la restricción de dominios de correo electrónico filtra las direcciones de correo electrónico y los dominios a los que Analytics envía marcadores, informes descargables y alertas. La lista de filtro de correo electrónico admite hasta 100 entradas, y cada una de ellas puede ser una dirección o un dominio completo de correo electrónico.
